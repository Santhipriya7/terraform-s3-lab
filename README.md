# Terraform AWS S3 Bucket 🚀

This project demonstrates how to create and manage an AWS S3 bucket using Terraform.

## 📌 Overview

* Create S3 bucket using Infrastructure as Code (Terraform)
* Enable versioning
* Configure server-side encryption
* Use variables and outputs

## 🛠️ Tools Used

* Terraform
* AWS CLI
* Git & GitHub

```

## ⚙️ Setup Instructions

### 1. Initialize Terraform

```
terraform init
```

### 2. Validate Configuration

```
terraform validate
```

### 3. Plan Infrastructure

```
terraform plan
```

### 4. Apply Changes

```
terraform apply
```

### 5. Destroy Resources (Cleanup)

```
terraform destroy
```

## 🔐 Notes

* Ensure AWS credentials are configured using `aws configure`
* Bucket name must be globally unique

Successfully created and managed an S3 bucket using Terraform.
